# 研究室におけるGitHubの活用

## GitHubができること

研究活動を進めるうえで，研究の透明性や再現性を確保することが必要です．そのためには，研究データやソースコードを適切に保存し，共有する仕組みが欠かせません．

:::tip
これから卒論研究や修士研究を進めるにあたり，研究データやソースコードを適切に管理することが重要です．
日頃から適切にデータを管理することで，パソコンの故障やデータの消失といったトラブルに備えることができます．

また，研究室内での共同作業を円滑に進めるためには，データやドキュメントの共有・管理がスムーズに行える環境を整える必要があります．
:::

Githubはコードを管理するためのクラウドベースのプラットフォームです．GitHubを利用することで，以下のようなことができます．

- 研究データの保存及び公開
- ソースコードや論文のバージョン管理
- 共同作業

その他にも，GitHubのアカウントを作成することで，個人サイトを[GitHub Pages](https://pages.github.com/)での公開や，[GitHub Copilot](https://docs.github.com/en/copilot)の利用が可能になります．
