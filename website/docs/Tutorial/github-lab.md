# 研究室におけるGitHubの活用

## GitHubができること

研究活動を進めるうえで，研究の透明性や再現性を確保することは非常に重要です．そのためには，研究データやソースコードを適切に管理し，共有する仕組みが欠かせません．また，パソコンの故障やデータの消失といったトラブルに備えて，日頃からバックアップを取ることも大切です．そして，研究室内での共同作業を円滑に進めるためには，データやドキュメントの共有・管理がスムーズに行える環境を整える必要があります．

GitHubを利用することで，以下のようなことが可能になります．

- 研究データの保存及び公開
- バージョン管理
  - ソースコード
  - LaTeXで作った論文
- 共同作業
- ドキュメント（README.md）の管理および公開

その他にも，GitHubのアカウントを作成することで，個人サイトを[GitHub Pages](https://pages.github.com/)での公開や，[GitHub Copilot](https://docs.github.com/en/copilot)の利用が可能になります．

## GitHubアカウントの作成

1. [GitHub](http://github.com/)にアクセス
2. 「Sign up」をクリックしてアカウントを作成

GitHubアカウントを作成したら，アカウントのUsernameやEmailを劉に知らせてください．
[研究室のGitHub Organization](https://github.com/isl-hosei)に招待します．

## GitHub Desktopのインストール

GitHub DesktopはGitHubを利用するためのアプリケーションです．Gitコマンドを使わずにGUIでGitHubを利用することができます．GitHub Desktopのウェブサイトは[こちら](https://desktop.github.com/)です．
